Códigos de Estado HTTP

1xx - Respuestas Informativas
Código Nombre Descripción Detalles
100 Continue Continuar El servidor ha recibido los headers de la petición y el cliente debería proceder a enviar el cuerpo de la misma.
101 Switching Protocols Cambio de Protocolo El servidor acepta el cambio de protocolo solicitado por el cliente (ej. de HTTP a WebSocket).
102 Processing Procesando El servidor está procesando la petición pero aún no hay respuesta disponible.
103 Early Hints Primeras Pistas Permite al cliente precargar recursos mientras el servidor prepara una respuesta.
2xx - Respuestas Satisfactorias
Código Nombre Descripción Detalles
200 OK Correcto La petición ha tenido éxito. El significado depende del método HTTP usado.
201 Created Creado La petición ha sido completada y ha resultado en la creación de un nuevo recurso.
202 Accepted Aceptado La petición ha sido aceptada para procesamiento, pero el procesamiento no ha sido completado.
203 Non-Authoritative Information Información No Autorizada El servidor procesó correctamente la petición, pero está devolviendo información de otra fuente.
204 No Content Sin Contenido La petición se ha completado con éxito pero no hay cuerpo de respuesta para devolver.
205 Reset Content Reiniciar Contenido El servidor ha completado la petición y el cliente debería reiniciar la vista del documento.
206 Partial Content Contenido Parcial El servidor entrega solo parte del recurso debido a un header Range enviado por el cliente.
207 Multi-Status Multi-Estado Respuesta WebDAV que puede contener múltiples códigos de respuesta.
208 Already Reported Ya Reportado Usado en WebDAV para evitar enlistar miembros internos de colecciones múltiples veces.
226 IM Used IM Usado El servidor ha cumplido la petición GET para el recurso y la respuesta es una representación del resultado tras aplicar una o más transformaciones.
3xx - Redirecciones
Código Nombre Descripción Detalles
300 Multiple Choices Múltiples Opciones Existen múltiples opciones para el recurso desde las cuales el cliente puede elegir.
301 Moved Permanently Movido Permanentemente El recurso se ha movido permanentemente a una nueva URL.
302 Found Encontrado El recurso se encuentra temporalmente en una URL diferente.
303 See Other Ver Otro La respuesta a la petición se encuentra en una URL diferente y debe ser obtenida mediante un método GET.
304 Not Modified No Modificado El recurso no ha sido modificado desde la última petición. Se usa para caché.
307 Temporary Redirect Redirección Temporal El recurso se encuentra temporalmente en otra URL. El método HTTP no debe cambiar.
308 Permanent Redirect Redirección Permanente El recurso se ha movido permanentemente a otra URL. El método HTTP no debe cambiar.
4xx - Errores del Cliente
Código Nombre Descripción Detalles
400 Bad Request Petición Incorrecta El servidor no pudo entender la petición debido a una sintaxis inválida.
401 Unauthorized No Autorizado La petición requiere autenticación del usuario.
402 Payment Required Pago Requerido Reservado para uso futuro. Podría usarse para servicios digitales de pago.
403 Forbidden Prohibido El servidor entendió la petición, pero se niega a autorizarla.
404 Not Found No Encontrado El servidor no puede encontrar el recurso solicitado.
405 Method Not Allowed Método No Permitido El método solicitado es conocido por el servidor pero no está soportado para el recurso.
406 Not Acceptable No Aceptable El recurso solicitado no puede generar contenido según los encabezados Accept enviados.
407 Proxy Authentication Required Autenticación de Proxy Requerida Similar al 401, pero indica que el cliente debe autenticarse primero con el proxy.
408 Request Timeout Tiempo de Espera Agotado El servidor agotó el tiempo de espera mientras esperaba la petición.
409 Conflict Conflicto La petición no pudo completarse debido a un conflicto con el estado actual del recurso.
410 Gone Ya no Disponible El recurso solicitado ya no está disponible en el servidor y no hay dirección alternativa.
411 Length Required Longitud Requerida El servidor rechaza la petición porque el campo Content-Length no está definido.
412 Precondition Failed Precondición Fallida El servidor no cumple una de las precondiciones indicadas por el cliente.
413 Payload Too Large Carga Útil Demasiado Grande La petición es más grande de lo que el servidor está dispuesto a procesar.
414 URI Too Long URI Demasiado Largo La URI solicitada es más larga de lo que el servidor está dispuesto a interpretar.
415 Unsupported Media Type Tipo de Medio No Soportado El formato multimedia de los datos solicitados no está soportado.
416 Range Not Satisfiable Rango No Satisfactorio El rango especificado en el campo Range del encabezado no se puede cumplir.
417 Expectation Failed Expectativa Fallida El servidor no puede cumplir con las expectativas indicadas en el campo Expect.
418 I'm a teapot Soy una Tetera El servidor se niega a intentar preparar café con una tetera (código de broma del 1 de abril).
422 Unprocessable Entity Entidad No Procesable La petición estaba bien formada pero no se pudo seguir debido a errores semánticos.
429 Too Many Requests Demasiadas Peticiones El cliente ha enviado demasiadas peticiones en un período de tiempo dado.
5xx - Errores del Servidor
Código Nombre Descripción Detalles
500 Internal Server Error Error Interno del Servidor El servidor ha encontrado una situación que no sabe cómo manejar.
501 Not Implemented No Implementado El servidor no soporta la funcionalidad requerida para completar la petición.
502 Bad Gateway Puerta de Enlace Incorrecta El servidor, mientras actúa como puerta de enlace, recibió una respuesta inválida del servidor ascendente.
503 Service Unavailable Servicio No Disponible El servidor no está listo para manejar la petición. Causas comunes: mantenimiento o sobrecarga.
504 Gateway Timeout Tiempo de Espera de la Puerta de Enlace El servidor, mientras actúa como puerta de enlace, no recibió una respuesta oportuna del servidor ascendente.
505 HTTP Version Not Supported Versión de HTTP No Soportada El servidor no soporta la versión del protocolo HTTP usada en la petición.
506 Variant Also Negotiates La Variante También Negocia El servidor tiene un error de configuración interna: la negociación transparente de contenido para la petición resulta en una referencia circular.
507 Insufficient Storage Almacenamiento Insuficiente El servidor no puede almacenar la representación necesaria para completar la petición.
508 Loop Detected Bucle Detectado El servidor detectó un bucle infinito mientras procesaba la petición.
510 Not Extended No Extendido Extensiones adicionales de la petición son requeridas para que el servidor la cumpla.
511 Network Authentication Required Autenticación de Red Requerida El cliente necesita autenticarse para ganar acceso a la red.
